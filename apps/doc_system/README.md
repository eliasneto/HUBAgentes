# Modulo Doc System

Este diretorio concentra a estrutura base do modulo `doc_system` dentro de `apps/`.

Objetivo:
- organizar recursos internos ligados a documentacao do sistema;
- separar a area documental por assunto;
- permitir expansao futura sem misturar com outros modulos.

Estrutura inicial:
- `gerenciar_agentes/`

Subarea preparada:
- `gerenciar_agentes/models.py`
- `gerenciar_agentes/views.py`
- `gerenciar_agentes/urls.py`
