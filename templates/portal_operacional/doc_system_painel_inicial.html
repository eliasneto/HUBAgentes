Painel inicial

Guia operacional

Como usar a tela inicial do portal como ponto de entrada, navegacao e leitura rapida das areas principais.

Esta pagina documenta o painel inicial do portal. Ela explica o objetivo da tela, o papel dos cards de acesso rapido e como o usuario deve usar esse painel como mapa geral do sistema.

DOC painel

Objetivo

Para que serve o painel inicial

  • Ser a porta de entrada principal do portal depois do login.
  • Apresentar rapidamente as areas operacionais e administrativas.
  • Permitir navegacao direta para os modulos mais importantes sem depender de menu lateral apenas.
  • Funcionar como uma visao geral para usuarios que estao se orientando no sistema.

Fluxo recomendado

Como um usuario costuma usar o painel inicial

  1. Entrar no sistema e confirmar que o login foi realizado com sucesso.
  2. Escolher o card da area desejada.
  3. Navegar para a tela operacional ou administrativa correspondente.
  4. Voltar ao painel quando quiser trocar rapidamente de contexto.

Observacoes importantes

Cuidados ao interpretar essa tela

  • O painel inicial e um ponto de orientacao, nao uma tela de execucao detalhada.
  • Alguns cards representam areas operacionais, outros representam areas administrativas.
  • Se uma area ainda estiver em evolucao, o painel pode apontar para uma tela reservada ou prototipada.
  • O menu lateral continua sendo a navegacao principal persistente, enquanto os cards funcionam como atalhos.
