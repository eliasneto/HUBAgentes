Usuarios e acessos

Guia operacional

Como cadastrar usuarios, organizar grupos e controlar quem pode operar cada parte do portal.

Esta tela documenta o modulo de usuarios e acessos. Ela explica o que aparece na listagem, os campos do cadastro, os papeis de acesso e os cuidados ao editar status, grupos ou senha.

DOC acessos

Passo a passo

Fluxo recomendado para criar ou ajustar um usuario

  1. Abra Administrador > Usuarios e acessos.
  2. Use Novo usuario para criar uma conta ou Editar usuario para ajustar uma existente.
  3. Preencha identificacao, papel principal e grupos.
  4. Defina se a conta esta ativa, se possui acesso tecnico e se e administradora total.
  5. Preencha senha apenas quando estiver criando a conta ou trocando a senha atual.
  6. Salve e revise a listagem para confirmar status, grupos e papel exibidos.

Perfis de acesso

O que cada perfil pode fazer no sistema

O sistema possui tres niveis de acesso controlados por grupos do Django. Adicione o usuario ao grupo correspondente ao seu papel na operacao.

Operador

Usuario logado sem grupo especial. Pode visualizar e executar agentes disponiveis. Nao pode criar, editar ou configurar agentes.

Grupo no Django: nenhum (qualquer usuario autenticado).
Analista

Pode criar, editar e executar agentes. Tambem acessa a tela Gerenciar agentes. Nao acessa configuracoes do sistema (integracoes, usuarios, custos).

Grupo no Django: analista
Administrador

Acesso completo: cria agentes, configura integracoes, fontes de documentos, usuarios, custos, tela de login e configuracoes gerais do sistema.

Grupo no Django: administrador ou flag superusuario.

Observacoes importantes

Cuidados ao editar usuarios e permissoes

  • Senhas existentes nao aparecem na tela e nao devem ser expostas na listagem.
  • Desativar um usuario impede o acesso ao portal, mas nao apaga a conta.
  • Marcar Administrador total amplia bastante os privilegios, entao use com criterio.
  • O proprio administrador nao deve remover seu proprio acesso critico sem validacao cuidadosa.
  • Para criar um Analista: cadastre o usuario e adicione-o ao grupo analista.
  • Para criar um Administrador: cadastre o usuario, adicione ao grupo administrador ou marque como superusuario.
  • Usuarios sem grupo ficam automaticamente como Operadores.
